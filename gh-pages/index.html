Motivation

At my company, Silverpoint, our designs often call for the use of shuffling panes of content, images or both. We've found that using M. Alsup's jQuery Cycle Plugin works phenomenally, but not for more than one item at a time. Each developer would spend time futzing with our product or JS until things were good enough, and then call cycle. This changes the game for us; perhaps you'll find it useful, too.

Basic Demo Grouping of paragraphs

Here, we'll just show what happens when you run group with no arguments whatsoever.

This is being done by running the following jQuery:

$('#basic_demo').find('p').group();

Advanced Demo Using jquery.group.js' settings

Now let's get a little trickier. Here are the whopping five settings provided to you by jquery.group.js:

$('#advanced_demo').find('p').group('article', true, '.set', '#item_', 3);
Note: You can pass these settings in any order you wish. You're welcome.

In the above code, these settings refer to, in order:

  • the html element you're wrapping things with
  • whether or not we want first-, last- and only-child classes placed on the items we're grouping'
  • the class of the wrapping tags
  • the prefix of the ID of the wrapping tags (the plugin appends numbers to this to keep IDs unique)
  • the size of the groups

While I felt it was important to allow the specification of any number of settings in any order, you can still pass jquery.group.js an object of settings. If you prefer a bit more order and structure in your life, try this:

$('#advanced_demo').find('p').group({
    'size': 3,
    'elem': 'article',
    'elem_class': 'set',
    'id_prefix': 'item_',
    'classing': true
});

<div id="advanced_demo_before">
  <p>This is paragraph One.</p>
  <p>This is paragraph Two.</p>
  <p>This is paragraph Three.</p>
  <p>This is paragraph Four.</p>
  <p>This is paragraph Five.</p>
</div>

	<article class="set" id="item_0">
		<p class="first-child">This is paragraph One.</p>
		<p>This is paragraph Two.</p>
		<p class="last-child">This is paragraph Three.</p>
	</article>
	<article class="set" id="item_1">
		<p class="first-child">This is paragraph Four.</p>
		<p class="last-child">This is paragraph Five.</p>
	</article>
</div>
